# Controls ::title

## Movement

| Key | Action |
|-----|--------|
| **Arrow keys** | Move / attack |
| **h j k l** | Move left / down / up / right |
| **y u b n** | Move diagonally |
| **o** | Autoexplore |

## Stairs

| Key | Action |
|-----|--------|
| **Shift + >** | Descend stairs |
| **Shift + <** | Ascend stairs / exit dungeon |

## Items

| Key | Action |
|-----|--------|
| **q** | Quaff potion |
| **r** | Read scroll |
| **w** | Wield weapon |
| **Shift + W** | Wear armour |
| **i** | Inventory |

## Other

| Key | Action |
|-----|--------|
| **x** | Examine mode |
| **Shift + S** | Save game |
| **?** | This help screen |
| **Escape** | Close / back |
